## Hello, world!

* Create a file named `main.rs` and enter the following `Hello, world!` code :
```rust
fn main() {
    println!("Hello, world!");
}
```

* Compile the `main.rs` source file :
```
rustc main.rs
```

* Launch the program :
```
./main
```
